# GeeGrow_SSD1306_128x32

This library is specially written to provide communication with board GeeGrow
0.96" I2C OLED 128x32 Display (SSD1306). The display can be purchased in our
store https://geegrow.ru . The library can use both standard I2C library Wire
and improved I2C library I2CTransport (https://github.com/geegrow/I2CTransport).

For more information you can visit https://github.com/geegrow/GeeGrow_ILI9341
